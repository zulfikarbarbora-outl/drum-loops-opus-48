# drum-loops-opus-48

Industrial drum loops at varied BPMs for beat-making. Sourced from Freesound packs 42294 + 42296 (looplicator Industrial Drum Loops, CC-BY-4.0).

## About

This repo holds **48 kbps Opus previews** for fast streaming in the web DAW.
Opus at 48k delivers near-CD quality at ~35% the size of MP3 128k.

- **Library**: Drum Loops (Industrial)
- **Files**: 7 Opus previews
- **Bitrate**: 48 kbps VBR
- **Container**: Ogg Opus (.opus)
- **License**: CC-BY-4.0
- **Source**: https://freesound.org/people/looplicator/
- **Branch**: `main`
